# Repo Analysis Notes

Bu klasor, repoyu calistirmadan yapilan kod okuma ve mimari analiz notlarini toplar.

Notlarin kapsami:

- Sadece dosya ve kaynak kod okuma
- Uygulama, test, build, install veya script calistirmadan inceleme
- Onemli mimari sinyaller, tasarim tercihleri ve riskli noktalarin not edilmesi

Ana dosyalar:

- `PROJECT-LOGIC-EXPLAINED.md`: tum projenin mantigini tek dosyada sade dille anlatan ana belge
- `00-overview.md`: reponun ne oldugu, yapisi ve temel gercekler
- `01-architecture.md`: kod seviyesinde mimari akis ve baglantilar
- `02-risks-and-observations.md`: kesinlesen riskler ve dikkat edilmesi gereken noktalar
- `03-confirmed-facts.md`: sadece kaynak koddan dogrudan dogrulanan bilgiler
- `04-reference-data-map.md`: `reference_data` klasoru ve snapshot veri yapisi
- `05-module-surface-map.md`: moduller, alt paketler ve placeholder yuzey haritasi
- `06-test-surface.md`: tek test dosyasinin kapsadigi alanlar
- `07-root-file-shape.md`: kok Python dosyalarinin satir ve import haritasi
- `07-cli-command-map.md`: tum subcommand'ler ve baglandiklari moduller
